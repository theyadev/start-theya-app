# start-theya-app

## Scaffold a NextJS or Nuxt3 project

With NPM:

```bash
$ npx start-theya-app
```

Then follow the prompts !

### Frameworks/Modules included
- NextJS
    - TailwindCSS
    - Mongoose
    - Redis (PLANNED)
    - SocketIO (PLANNED)
- Nuxt3 (PLANNED)
    - TailwindCSS (PLANNED)
    - Mongoose (PLANNED)
    - Redis (PLANNED)
    - SocketIO (PLANNED)
